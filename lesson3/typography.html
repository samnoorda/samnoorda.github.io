Font's That I will use:

  • 'Libre Baskerville' will be used for the headings and special markings on the page. I chose this font because I feel like it can get the reader's attention without being too obnoxious.
  • 'Quicksand' will be used primarily for all of the body text of the web page. I like this font because it has a subtle elegance to it.
