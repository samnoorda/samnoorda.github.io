P.A.R.C. Alignment

Amazon

Amazon.com

Amazon uses alignment in the top banner that has all of the links to different sections of its website(i.e. Deals, Whole Foods, etc.). The user knows that each of these words will take them to a different section of the website if they click on them.


P.A.R.C. Repetition

Sorel

Sorel.com

Sorel uses Repetition to show that each shoe and each color of the selected shoe will do the same thing if the user clicks on each of them. Every item in the list is equal to the others.


P.A.R.C. Contrast

IMDb

IMDb.com

IMDb uses contrast by using lighter colors in for their text and a dark color for their background. This greatly enhances readability throughout their website.
